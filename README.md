## 📚 Sobre el Curso

El curso ofrece una visión integral de los principales enfoques, modelos y herramientas de la IA moderna, combinando teoría rigurosa con práctica intensiva en programación.

### Temas Cubiertos

- 🔍 **Introducción a IA**: Historia, conceptos fundamentales y aplicaciones
- 🖼️ **Procesamiento Digital de Imágenes**: Adquisición, filtrado y operaciones
- 📝 **Procesamiento de Lenguaje Natural (NLP)**: Tokenización, NER, análisis de corpus
- 🧠 **Sistemas Expertos**: Representación del conocimiento e inferencia
- 🕸️ **Redes Neuronales Artificiales**: Perceptrón, backpropagation, optimización
- 🤖 **Machine Learning**: Métodos supervisados, no supervisados y por refuerzo
- 🚀 **Deep Learning**: CNNs, RNNs, LSTM, transfer learning
- 🧬 **Algoritmos Genéticos**: Evolución y optimización combinatoria
- 🌊 **Lógica Difusa**: Conjuntos difusos e inferencia aproximada
- 📊 **Redes Bayesianas**: Inferencia probabilística
- 🗺️ **Planificación y Búsqueda**: Algoritmos clásicos de búsqueda y planificación

## 🎯 Cómo Navegar por el Repositorio

### Para Acceder a Trabajos Específicos

- **Laboratorios**: Consulta la carpeta `Laboratorios/` para ver todos los talleres y tareas prácticas
- **Exposiciones**: Revisa `Exposicion/` para acceder a las presentaciones de artículos
- **Materiales de Clase**: En cada carpeta de taller hay materiales complementarios

### Para Ejecutar los Notebooks

1. Clona o descarga el repositorio
2. Navega a la carpeta del taller de interés
3. Abre el archivo `.ipynb` en Jupyter Notebook o JupyterLab
4. Ejecuta las celdas para ver los resultados

## 🔧 Herramientas y Dependencias

El proyecto utiliza principalmente:

- **Python 3.8+**
- **Jupyter Notebook** / JupyterLab
- **NumPy, Pandas, Scikit-learn**
- **TensorFlow / PyTorch**
- **NLTK** (procesamiento de lenguaje natural)
- **OpenCV** (visión por computadora)
- **Matplotlib, Seaborn** (visualización)

Para instalar dependencias:
```bash
pip install -r requirements.txt
```
